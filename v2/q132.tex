  There is a function $f$ so that $f(0) = 0$ and when $h$ is small, $f^{-1}(h)$ is close to $10 h$.  What is a good guess for $f'(0)$?

    \choice[correct]{$1/10$}
    \choice{$-1/10$}
  \end{multipleChoice}
